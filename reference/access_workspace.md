# Access a workspace

Provides access to a workspace.

## Usage

``` r
access_workspace(login, ws_id = NULL, verbose = FALSE)

# S4 method for class 'LoginStudio'
access_workspace(login, ws_id = NULL, verbose = FALSE)

# S4 method for class 'LoginTestcenter'
access_workspace(login, ws_id = NULL, verbose = FALSE)
```

## Arguments

- login:

  [Login](https://iqb-research.github.io/eatPrepTBA/reference/Login-class.md)
  Login object generated by either
  [`login_studio()`](https://iqb-research.github.io/eatPrepTBA/reference/login_studio.md)
  or
  [`login_testcenter()`](https://iqb-research.github.io/eatPrepTBA/reference/login_testcenter.md).

- ws_id:

  Numeric. Workspace id (see URL or call login function with
  `verbose = TRUE`).

- verbose:

  Logical. If `TRUE`, additional information is printed. Defaults to
  `FALSE`.

## Value

[Workspace](https://iqb-research.github.io/eatPrepTBA/reference/Workspace-class.md)
object.

## Functions

- `access_workspace(LoginStudio)`: Provide access to selected workspaces
  after logging in

- `access_workspace(LoginTestcenter)`: Provide access to a selected
  workspace after logging in
