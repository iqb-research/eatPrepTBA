# Extract metadata profiles from units with metadata

This function extracts the unit metadata profiles of a unit table
generated by
[`add_metadata()`](https://iqb-research.github.io/eatPrepTBA/reference/add_metadata.md).
This can be used for converting character labels to factors.

## Usage

``` r
extract_metadata(units)
```

## Arguments

- units:

  Tibble holding units retrieved from
  [`get_units()`](https://iqb-research.github.io/eatPrepTBA/reference/get_units.md)
  and prepared with
  [`add_metadata()`](https://iqb-research.github.io/eatPrepTBA/reference/add_metadata.md).

## Value

A list of three tibbles, `ws_settings`, `unit_md_profile`, and
`item_md_profile`.
